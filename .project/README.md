# Project Guide

## What This Project Is

This repository contains `gentms`, a Python CLI that generates fiscal-year timesheet Excel workbooks.

## Where To Start

- Read `README.md` for end-user installation and CLI usage.
- Read `.plans/gentms-cli.md` for the current implementation plan.
- Read `.decisions/001-python-openpyxl-cli.md` for the stack and naming decision.
- Read `.project/build.md` and `.project/testing.md` before changing code or verification.

## Minimum Setup

- Python 3.11 or later
- Install dependencies with `python3 -m pip install -e .[dev]`
