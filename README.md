# gentms

`gentms` は、年度ベースの勤怠入力用 `xlsx` ファイルを生成する CLI ツールです。

## できること

- 4月から翌年3月までの 12 シートを持つ年度ファイルを生成
- 指定した月だけの月別ファイルを生成
- 1月から3月を自動で翌年として扱う
- `日付` 列を Excel の日付セルとして保持しつつ、表示は日だけにする
- 時刻と休暇種別にプルダウンを設定する
- 重複月や既存ファイル・既存ディレクトリがある場合はエラーにする

## 動作要件

- Python 3.11 以上

## セットアップ

```bash
python3 -m pip install -e '.[dev]'
```

## 使い方

年度ファイルを生成:

```bash
gentms annual 2026
```

年度ファイルの短縮形:

```bash
gentms 2026
```

年度ファイルをディレクトリ付きで生成:

```bash
gentms annual 2026 --create-dir
```

指定月のファイルを生成:

```bash
gentms monthly 2026 12 1 2 3
```

指定月のファイルをディレクトリ付きで生成:

```bash
gentms monthly 2026 12 1 2 3 --create-dir
```

## 出力ルール

- `annual` は `YYYY年度(RN).xlsx` を生成
- `monthly` は実際の年月に合わせた `YYYY-MM.xlsx` を生成
- `--create-dir` 指定時は `YYYY年度(RN)/` 配下に出力
- 既存ファイルや既存ディレクトリがある場合はエラー

`2026` 年度の例:

- 年度ファイル: `2026年度(R8).xlsx`
- 出力ディレクトリ: `2026年度(R8)/`
- `12` 月: `2026-12.xlsx`
- `1` 月: `2027-01.xlsx`

## ブック仕様

各シートの列は以下です。

- `日付`
- `出勤 (時)`
- `出勤 (分)`
- `退勤 (時)`
- `退勤 (分)`
- `休暇種別`

プルダウンの候補:

- `出勤 (時)`, `退勤 (時)`: `6` から `22`
- `出勤 (分)`, `退勤 (分)`: `00`, `10`, `20`, `30`, `40`, `50`
- `休暇種別`: `年休`, `忌引`
